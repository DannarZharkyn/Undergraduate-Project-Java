# Somni Website

A simple static website for Somni. It can be opened directly in a browser, hosted as a public link, and paired with a QR code.

Live site:

```text
https://dannarzharkyn.github.io/
```

## Files

- `index.html` is the website.
- `styles.css` controls the design.
- `assets/somni-hero.svg` is the hero image.
- `assets/somni-qr.png` is the QR code after publishing.
- `qr.html` can generate a QR code preview after you paste the hosted website URL.

## Open Locally

Open `index.html` in a browser.

## Publish

This repository is configured for GitHub Pages from the `main` branch with the included Pages workflow. The public URL is:

```text
https://dannarzharkyn.github.io/
```

The QR code in `assets/somni-qr.png` points to that URL.

## Customize

Update the text in `index.html`, especially the contact email:

```html
hello@somni.ventures
```

Replace it with your real venture email before publishing.
